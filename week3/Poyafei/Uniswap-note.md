# 概念

## 基本概念

Uniswap是一个基于以太坊的**去中心化交易所（DEX）协议**，它允许用户无需中介即可直接交换加密货币。其核心是一个[自动做市商（AMM）]模型，通过智能合约和流动性池进行自动化交易，而非传统的订单簿。除了交易，用户还可以通过向流动性池提供资金来赚取交易费用，并可以通过其原生代币UNI参与治理投票。 

## 理解

UNI/ETH，卖出ETH，买入UNI，或者卖出UNI，买入ETH，要与程序完成这笔交易，合约本身首先要持有这两种资产，比如100个ETH和2000个UNI，这样，交易员用1个ETH买入20个UNI后，合约持有资产可能变成101个ETH和1980个UNI。如果另一个交易员稍后卖出40个UNI，合约持有的资产又可能变成2020个UNI和99个ETH。

# 理论

## 机制

### 自动化做市商 AMM

1. **概念：**为了提高市场的流动性，交易所通常会引入做市商，进行价格操作。AMM 采用智慧合约来运作由代币组成的**流动性池（liquidity pools）**，用户直接与这些合约互动完成交易。
2. **AMM类别：**
    
            恒定函数做市商（CFMM）
    
    - 恒定乘积（Uniswap）：x * y = k（X 和 Y代表流动性池中两种资产的单位数量）
    - 恒定和 （CSMM）： x + y = k

### **恒定乘积公式**

1. **概念：**利用恒定乘积公式（x * y = k）来计算资产的价格。在此公式中，x 代表交易资产的储备，y 代表定价资产的储备。当从池中取出（买入）代币时，必须存入（售出）一定比例的金额，以保持 k 恒定。池中代币的比率决定了代币的价格。
2. **核心公式**：x×y=k

### 流动性提供者LP（Liquidity Provider）

1. **概念：**流动性供应商(LP)向流动性资金池注入资金。您可以将流动性资金池视为一大笔资金，交易者可以与这些资金进行交易。作为向协议提供流动性的回报，流动性供应商会从交易池开展的交易中赚取费用。
2. **原理：**把两种token 按比例存进去，拿到LP Token 当凭证。

### 手续费模型

1. **概念：**在大多数去中心化交易协议（如Uniswap）中，每笔交易都会产生一部分手续费。这些手续费通常直接分配给提供流动性的LPs，以激励他们持续为交易对提供深度。去中心化交易平台的手续费主要包括网络费用和平台本身的交易费用。平台手续费一般为0.1%~0.3%。
2. **计算：**
    
    在 Uniswap V2 中，手续费分为两种。用户在 Swap 时只需支付 0.3% 的手续费，手续费的六分之五给流动性提供者，六分之一给协议。
    

### swap

1. **概念：**指使用智能合约在不同代币之间进行交换的过程，允许用户无需传统中介即可直接交换资产。
2. **用法：**想要用代币 𝑥 兑换代币 𝑦 时，系统会根据恒定乘积公式自动计算兑换率。
3. **返还方式**：LP 在移除流动性时，才会实际收到他们应得的这部分手续费。他们取回的代币数量会基于他们对池子总流动性所占的比例，同时他们所持有的 LP Token 会被销毁。

### **去中心化价格预言机（或称为预言机）**

1. **概念**：它通过分散的节点网络从多个外部来源获取、验证和传递数据，以确保数据的准确性和可靠性，从而解决单点故障和数据不准确的风险，并使智能合约能够基于真实世界的信息做出反应和执行。
2. **功能：**去中心化预言机充当数据提供者，将外部数据（如价格、天气信息）安全地传输到链上。

## 数据流

用户 -> Router -> Factory -> Pair -> **Library**(计算)

- Factory 合约是 Uniswap-v2 用来创建资金池的合约。
- Pair 合约是 Uniswap-v2 的资金池的合约。
- Router 合约为用户提供了一个面向用户的智能合约，用于
    - 安全地 [兑换成对代币](https://learnblockchain.cn/article/11282)
    - 安全地 [铸造和销毁 LP 代币](https://learnblockchain.cn/article/11266)（添加和移除流动性）
    - 通过与包装以太（WETH）ERC20 合约集成，增加了兑换 Ether 的能力。
    - 增加了核心合约中遗漏的与 [滑点](https://learnblockchain.cn/article/11255) 相关的安全检查。
    - 添加了对转账费用代币的支持
